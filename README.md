# ListViewSwiftUI


A simple list showcasing SwiftUI List/ListView types of gradients in iOS using LinearGradient, RadialGradient, and AngularGradient.


A minimal SwiftUI demo project showcasing how to build and display a simple list using `List` and local data model.

## Overview

This project demonstrates the basics of SwiftUI list rendering, including:
- Displaying a collection of data in a `List`
- Simple UI composition in SwiftUI
- Detail View with data passing from list via use of NavigationLink
=
It is intended for beginners who want to understand how list-based views work in SwiftUI.

## Features

- Static sample data displayed in a list
- SwiftUI `List`
- Clean and minimal UI
- Passing data in Detail View
- Lightweight and easy to extend

## Requirements

- Xcode 16 or later
- iOS 13+ (or as configured in the project)
- Swift 6
